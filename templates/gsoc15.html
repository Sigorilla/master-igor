Project Proposal

Title

Interactive Tutorials for jQuery Learning Center

Description

This project will be a repository on GitHub, for which you can create folders with different levels of difficulty. Each folder will include the tasks as HTML pages with video (from YouTube) and various training slides such as cards or short questions. You must correctly answer tasks for further reading the article. Also you will be able to take a hint or get a full answer.

The user will be required to write a necessary part of the existing code to subccessfully complete the task.

Some tasks (steps) can be grouped into a chain, so user will have to solve all the steps to complete this task.

More details how to get started you can see here: arschmitz/jquery-tutorials.
